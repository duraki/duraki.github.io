Subdomains

-- [0x1 - Steps]

    Use amass to gather passive data
    Use gobuster for bruteforce
    Use commonspeak2, SecList
    Run massdns on input
    Run altdns on the result set
    Run massdns on input from altdns results

-- [0x2 - Example]

Passive:
--
$ amass enum --passive -d [DOMAIN]
$ amass enum --passive -d [DOMAIN] -o amass.recon 			# Output to file

Bruteforce:
--
$ gobuster dns -d [DOMAIN] -w ~/util/commonspeak2/subdomains/subdomains.txt
$ gobuster dns -d [DOMAIN] -w [WORDLIST] -o gobuster.recon -t 30 # Output to file, Via 30 threads
