## Specifications

- SomeIP, Ethernet, IP Stack
- New standard 10BASE-T1S
- 10Mbit/s
- Half-duplex
- PLCA bus cycle - only specific PHY that can send data
- Physical Layer Collision Avoidance (PLCA)
- Avoid collision of data
- Multiple applications on single cable
